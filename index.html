Many developers use GitHub Copilot, Cursor, and Claude Code at the same time. You might lose track of your budgets. Quota runs on your computer. It monitors requests from your tools. It summarizes how much you spend each day, week, and month.

Follow these steps to set up the software on your Windows computer:

  1. Go to the download page.
  2. Find the file that ends in .exe for Windows.
  3. Click the file name to download it to your Downloads folder.
  4. Open your Downloads folder and double-click the file to start the installer.
  5. Follow the prompts on your screen to finish the setup.
  6. Launch the application from your Start Menu after installation finishes.

⚙️ Configuration

Quota starts monitoring automatically. Upon the first launch, the app detects your installed AI extensions. Some tools require you to link your account to provide usage data. Click the Settings icon inside the app to view your connected providers. If a tool does not show up, ensure it is active in your code editor.

🛡️ Privacy and Safety

Your data stays on your machine. Quota does not send your usage history or code snippets to external servers. The app stores all logs in a local database on your hard drive. This local-first design keeps your information private. You hold full control over your usage history at all times.

📈 Tracking your Usage

The main screen shows a graph of your daily usage. You can filter data by specific tools or time ranges. If you notice a spike in costs, you can hover over the data points to see which tool caused the increase. This transparency helps you manage your subscription plans for paid AI services.

🧩 Using the VSCode Extension

Quota includes a helper for your code editor. This extension allows you to see your current session usage without leaving your work. Install the Quota extension from the VSCode Marketplace to unlock this feature. It syncs with the desktop app to keep your data current.
